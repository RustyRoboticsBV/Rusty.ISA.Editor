# ActionGraph Editor Plugins

This directory contains all of ActionGraphs editor plugins:
- Editor scripts: all the graph editor functionality.
- Importer plugins:
  - Program Importer: a `.agxp` to `InstructionProgram` importer.
  - Node Definition Importer: a `.agxn` to `NodeDefinition` importer.
  - Instruction Definition Importer: a `.agxi` to `InstructionDefinition` importer.
